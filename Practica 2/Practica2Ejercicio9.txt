ORG 1000h

     DB 13
 CLAVE DB "1234"
 INTENTO DB ?
 MSJMAL DB "ACCESO DENEGADO"
 MSJBIEN DB "ACCESO PERMITIDO"
 FIN5 DB ?

ORG 2000h
 MOV BX, OFFSET MSJ
 MOV AL, OFFSET CLAVE - OFFSET MSJ
 INT 7
 
 MOV BX, OFFSET INTENTO
 INT 6
 
 MOV AL, INTENTO
 MOV BX, OFFSET CLAVE
 CMP AL,[BX]
 JNZ SIG2
 INC AH
 
 SIG2:MOV BX, OFFSET INTENTO
 INT 6
 
 MOV AL, INTENTO
 MOV BX, OFFSET CLAVE+1
 CMP AL,[BX]
 JNZ SIG3
 INC AH
 
 SIG3:MOV BX, OFFSET INTENTO
 INT 6
 
 MOV AL, INTENTO
 MOV BX, OFFSET CLAVE+2
 CMP AL,[BX]
 JNZ SIG4
 INC AH
 
 SIG4:MOV BX, OFFSET INTENTO
 INT 6
 
 MOV AL, INTENTO
 MOV BX, OFFSET CLAVE+3
 CMP AL,[BX]
 JNZ FIN
 INC AH
 
 FIN: CMP AH,4
 JNZ MAL
 MOV BX, OFFSET MSJBIEN
 MOV AL, OFFSET FIN5 - OFFSET MSJBIEN
 INT 7
 JMP FINAL
 
 MAL: MOV BX, OFFSET MSJMAL
 MOV AL, OFFSET MSJBIEN - OFFSET MSJMAL
 INT 7
 FINAL: INT 0
